## ⚠️ IMPORTANT RULES

- **ALWAYS clear all existing values in a textbox BEFORE typing a new value.** For every input field (username, password, etc.), first select all text (Ctrl+A / Cmd+A) and delete it, then type the new value. This prevents appending to pre-filled or stale values.

## Admin Login Flow

1. Navigate to `http://localhost:5173/admin/login`
2. For the **Email** field:
   - Click the email input field
   - Press Ctrl+A (or Cmd+A) to select all existing text
   - Press Backspace/Delete to clear
   - Type: `superadmin@gmail.com`
3. For the **Password** field:
   - Click the password input field
   - Press Ctrl+A (or Cmd+A) to select all existing text
   - Press Backspace/Delete to clear
   - Type: `superadmin20797`
4. Click the login/submit button
5. After successful login, navigate to the desired admin page

## Common Admin Pages

- **Dashboard**: `/admin/dashboard`
- **Add Post**: `/admin/posts/add`
- **Posts List**: `/admin/posts`
